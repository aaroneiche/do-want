Welcome to Do Want!

Do Want! is a Gift Registry system made to make it easier to manage gift exchanges in families and friends. Do Want is meant to be easy to install and administer so you can focus on the fun part of gift exchanges: Getting Gifts Giving gifts to others.

How Do I install Do Want?

Download Do want from this site and drop it into your web browser. Do Want will automatically load up the installer page and walk you through the setup process. We've tried to make it as simple as possible to get started with Do Want.

What are Do Want's system requirements?

Do Want has the following system requirements to utilize all it's features:

  • PHP 5.3 or greater
  • MySQL 5 or greater
  • A Web host of some kind
  • A Modern Web Browser (Generally we mean Chrome, Firefox, Safari, or IE8+)

I have a problem with Do Want, where can I get some help?

You can join the discussion list to get some clarification in the Do Want Google Group. If you have found a bug, or would like to suggest a feature, you can do it in the issues log.
